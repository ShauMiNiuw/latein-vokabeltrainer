Level 3: Übersetzung + Geschlecht, 15 % Tipps/Joker, 60 Sek./Frage

🖋️ Grammatikinformationen

Syntax:

Kurzformen wie Adv. oder Akk. DÜRFEN statt den ausgeschrieben Formen verwendet werden.

Trage NUR den Inhalt der zweiten Spalte ein (NICHT das Geschlecht).

Trage ALLE Formen wie im Buch ein, getrennt durch ein KOMMA (form1, form2, form3)

Trage bei Wörtern wie "eum" das Grammatikfeld so ein: eos (sie (maskulin Plural))

Trage bei Wörtern wie "ruinae" das Grammatikfeld so ein: ruinarum Plural

Trage bei Wörtern wie "navis" das Grammatikfeld so ein: navis (Genitiv Plural: navium)

Trage bei Wörtern wie "ipse" das Grammatikfeld so ein: ipsa, ipsum (Demonstrativpronomen)

Trage bei Wörtern wie "cuncti" das Grammatikfeld so ein: cunctae, cuncta Plural

Trage bei Wörtern wie "munus" das Grammatikfeld so ein: munus (Nominativ/Akkusativ Plural: munera)

Trage bei Wörtern wie "Gallus" das Grammatikfeld so ein: Galli galla, gallum

Trage bei Wörtern wie "unus" das Grammatikfeld so ein: una, unum (Genitiv: unius, Dativ: uni)

Trage bei Wörtern mit weiteren Formen in Form von Endungen (-a, -um) IMMER die ausgeschriebenen Wörter ein.
